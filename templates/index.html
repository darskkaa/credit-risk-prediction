About This System
🤖 AI-Powered Credit Default Risk Prediction Model

This is a comprehensive machine learning system that analyzes corporate financial data to predict the likelihood of credit default. Built with ensemble ML models (Logistic Regression, Random Forest, Gradient Boosting) and real-time financial data integration.

🧠 How It Works
  1. Data Input: Financial metrics are collected from user input or FMP API sources
  2. Feature Engineering: 22+ financial ratios and indicators are calculated automatically
  3. ML Prediction: Ensemble of 3 trained models analyze the data and generate risk probabilities
  4. Interpretability: Feature contribution analysis explains the ML model's decisions
Key Features
  • Real-time Analysis: Instant risk assessment
  • Multiple Data Sources: Manual input or FMP API integration
  • Comprehensive Metrics: 22+ engineered financial features
  • Visual Insights: Charts and graphs for better understanding
  • Professional Grade: Production-ready ML pipeline
🛠️ Technical Details
  • Framework: Flask web application with RESTful API
  • ML Models: Logistic Regression, Random Forest, Gradient Boosting
  • Feature Engineering: 22 financial ratios with normalization
  • Ensemble Method: Weighted average with confidence scoring
  • Data Source: Financial Modeling Prep (FMP) API
  • Visualization: Chart.js interactive charts
  • Architecture: Modular, scalable ML pipeline
